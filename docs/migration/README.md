# Документация по инфраструктуре и миграции

Все доки по переезду ФлорМажора в РФ-инфраструктуру для соблюдения 152-ФЗ. Четыре документа описывают **разные сценарии** разной зрелости — выбираем тот, который соответствует текущему состоянию.

## Что читать прямо сейчас

| Документ | Когда нужен |
|---|---|
| **[`overview.md`](overview.md)** | Общая логика миграции — Сценарий B (без формы лидов): почему именно так, что меняется в коде, юридический эффект, риски. Читать первым, чтобы понять «почему» вместо «как». |
| **[`cheap.md`](cheap.md)** | Текущий принятый сценарий: одна VM в Timeweb за ~1 100 ₽/мес + self-host Supabase в Docker Compose. Backbone-чек-лист на 17 разделов и 15 шагов. |
| **[Spec миграции (текущий)](../superpowers/specs/2026-05-02-cheap-migration-design.md)** | Принятые в брайнсторме уточнения к `cheap.md`: shrink Supabase под 4 ГБ, Studio on-demand, бэкапы 6 ч / 14 д, точный scope ветки `migrate-cheap-stack`, работа без SMTP. **Читать вместе с `cheap.md`.** |
